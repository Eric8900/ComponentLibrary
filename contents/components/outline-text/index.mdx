## Props

|Prop          |Type        |Description |Default Value |
| :------------ | :--------- | :---------- |:----------|
| <span className="highlight-text">children</span>   | <span className="highlight-text">ReactNode</span>    | The element of text that will go on the button | <span className="highlight-text">undefined</span> |
| <span className="highlight-text">from</span>   | <span className="highlight-text">string</span>    | Color you want the gradient to go from  | <span className="highlight-text">"\#000"</span> |
| <span className="highlight-text">to</span>   | <span className="highlight-text">string</span>    | Color you want the gradient to go towards  | <span className="highlight-text">"\#000"</span> |
| <span className="highlight-text font-extrabold text-red-500">top</span>   | <span className="highlight-text font-extrabold text-red-500">number</span>    | <span className="font-extrabold text-red-500">Use this number to adjust how far up or down the outline should go <br/>(The outline may be misaligned if you use different length words or text sizes)</span> | <span className="highlight-text font-extrabold text-red-500">15</span> |
